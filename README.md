# Readme
A classification of Interrupts.

### Interrupts

Interrupt Handlers要处理的Interrupts例如：
- 来自电源的Interrupts
  - 启动
  - 停止
- 来自处理器的Interrupts
  - 中止
  - 缺页
  - 系统调用
- 来自定时器的Interrupts
  - 心跳
- 来自外设的Interrupts
  - 中断
